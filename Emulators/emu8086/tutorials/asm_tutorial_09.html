The Stack

PUSH REG
PUSH SREG
PUSH memory

POP REG
POP SREG
POP memory


For every PUSH there must be a POP.


Examples: 



ORG    100h
MOV    AX, 1234h
PUSH   AX 
MOV    AX, 5678h
POP    AX
RET
END


ORG    100h
MOV    AX, 1212h
MOV    BX, 3434h
PUSH   AX
PUSH   BX
POP    AX
POP    BX
RET
END





PUSH X:
  • Subtract 2 from SP

  • Copy X to [SS:SP]

POP X:
  • Copy [SS:SP] to X

  • Add 2 to SP

SS:SP = top of the stack.
