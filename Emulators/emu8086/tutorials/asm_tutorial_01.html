8086 Central Processing Unit or CPU

Registers:
  • AX - accumulator
  • BX - base
  • CX - count
  • DX - data
  • SI - source index
  • DI - destination index
  • BP - base pointer
  • SP - stack pointer
  • CS - code segment
  • DS - data segment
  • ES - extra segment
  • SS - stack segment
  • IP - instruction pointer
  • FLAGS - status and results
