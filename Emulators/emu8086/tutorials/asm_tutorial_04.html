Interrupts


Syntax:
INT value


Example of INT 10h/0Eh sub-function: 

ORG    100h
MOV    AH, 0Eh  
MOV    AL, 'H'   
INT    10h     
MOV    AL, 'e'  
INT    10h    
MOV    AL, 'l' 
INT    10h 
MOV    AL, 'l' 
INT    10h 
MOV    AL, 'o'
INT    10h
MOV    AL, '!'
INT    10h
RET
