Memory Access





MOV instruction

  • first operand can be a register or memory location.

  • second operand can be a number, a register or a memory location.

  • both operands must be of the same size. 


ORG 100H           ; make simple program 
MOV AX, 0B800h     ; AX = B800 
MOV DS, AX         ; DS = AX 
MOV CL, 'A'        ; CL = 'A' 
MOV CH, 1101_1111b ; CH = colour code 
MOV BX, 15Eh       ; BX = 015E 
MOV [BX], CX       ;  [DS:BX] = CX 
RET                ; exit
