Variables


name DB value

name DW value


name - should start with a letter

value - can be a numeric value or text



Example: 

ORG 100h
MOV AL, var1
MOV BX, var2
RET 
VAR1 DB 7
var2 DW 1234h



Variable Arrays


A DB 48h, 65h, 6Ch, 6Ch, 6Fh, 00h

Square brackets are used to access individual elements:


MOV AL, A[3]
MOV AL, A[SI]

Legal memory pointers: BX, SI, DI, BP
