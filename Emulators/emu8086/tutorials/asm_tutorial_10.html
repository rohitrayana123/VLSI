Macro 

name    MACRO  [parameters,...]

             <instructions>

ENDM


Example: 

Bird    MACRO  p1, p2, p3
     MOV AX, p1
     MOV BX, p2
     MOV CX, p3
ENDM
ORG 100h
Bird 1, 2, 3
Bird 4, 5, DX
RET



LOCAL directive is used to avoid redeclaration errors:

cat MACRO
LOCAL label1, label2
CMP  AX, 2
JE label1
CMP  AX, 3
JE label2
label1:
  INC  AX
label2:
  ADD  AX, 2
ENDM
ORG 100h
cat
cat
RET
END
