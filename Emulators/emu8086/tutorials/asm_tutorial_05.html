Library of common functions - emu8086.inc

The default location for inc files is: \inc\


Macros:

  • PUTC char - prints out an ASCII char at current cursor position.

  • GOTOXY col, row - sets cursor position.

  • PRINT string - prints out a string.

  • PRINTN string - prints out a string and carriage return.

  • CURSOROFF - turns off the text cursor.

  • CURSORON - turns on the text cursor.

include "emu8086.inc"
ORG    100h
PRINT 'Hello World!'
GOTOXY 10, 5
PUTC 'T'
PUTC 'E'
PUTC 'S'
PUTC 'T'
RET
END



Procedures:

  • CALL PRINT_STRING - print a null terminated string at current cursor position, address of string in DS:SI Declare: DEFINE_PRINT_STRING before END directive.

  • CALL PTHIS - print a null terminated string at current cursor position. Zero terminated string should be defined just after. To use it declare: DEFINE_PTHIS

  • CALL GET_STRING - get a null terminated string from a user, the received string is written to buffer at DS:DI, buffer size should be in DX. Declare: DEFINE_GET_STRING

  • CALL CLEAR_SCREEN - clear the screen. Declare: DEFINE_CLEAR_SCREEN

  • CALL SCAN_NUM - get multi-digit signed number from the keyboard, and stores the result in CX register. Declare: DEFINE_SCAN_NUM

  • CALL PRINT_NUM - prints a signed number from AX register. Declare: DEFINE_PRINT_NUM and DEFINE_PRINT_NUM_UNS

  • CALL PRINT_NUM_UNS - prints out an unsigned number in AX register. Declare: DEFINE_PRINT_NUM_UNS 

include "emu8086.inc"
ORG    100h
CALL   pthis
DB     "Enter the number: ", 0
CALL   scan_num 
MOV    AX, CX 
CALL   pthis
DB     13, 10, "Your lucky number is: ", 0
CALL   print_num 
RET 
DEFINE_SCAN_NUM
DEFINE_PRINT_NUM
DEFINE_PRINT_NUM_UNS
DEFINE_PTHIS
END
